# MouthSpriteExtractor-SAM3 Dependencies
#
# Setup:
#   1. Create virtual environment (Python 3.12 required):
#      uv venv .venv --python 3.12
#
#   2. Activate:
#      .venv\Scripts\activate  (Windows)
#      source .venv/bin/activate  (Linux/Mac)
#
#   3. Install PyTorch with CUDA 12.6:
#      uv pip install torch torchvision --index-url https://download.pytorch.org/whl/cu126
#
#   4. Install dependencies:
#      uv pip install -r requirements.txt
#
#   5. Authenticate with HuggingFace (required for SAM3 model):
#      huggingface-cli login
#
#   6. Download sam3.pt from HuggingFace:
#      https://huggingface.co/facebook/sam3
#      Place sam3.pt in the project directory
#
# Note: SAM3 model requires access approval from HuggingFace
